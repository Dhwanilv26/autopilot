================================================================================
IMPORTANT STREAMING NOTES
================================================================================

1. "delta" contains partial updates.
2. "tool_calls" is an array.
3. "index" identifies which tool call the fragment belongs to.
4. "function.arguments" is streamed as STRING fragments.
5. You must concatenate all fragments.
6. After complete, parse with json.loads().
7. finish_reason == "tool_calls" signals completion.
